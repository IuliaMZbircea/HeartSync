Add Consultation

👤 Patient Information At the top of the page, you can see: Full Name, Sex, and CNP of the patient – for easy identification.

📤 Send Medical Referral

This section allows you to refer the patient to another specialist or doctor. You can:

  • Select a doctor from the dropdown list.
  • Choose a referral type.
  • Write the reason for the referral.

Once completed, click “Submit Referral” to send it.

💊 Add New Medication

Use this section to prescribe or register a new medication for the patient:

  • Fill in medication details: name, dosage, frequency, route, start and end dates, and prescribing doctor.
  • You can also add optional notes.
  • Click “Save Changes” to save the new medication to the patient's record.

📝 Validation ensures that all required fields are completed before submission.

➕ Add Diagnosis / Condition

Here, you can register a new medical diagnosis for the patient:

  • Use the diagnosis search component to select a medical condition.
  • The corresponding ICD code will be auto-filled.
  • Choose the category (chronic, genetic, etc.).
  • Add an optional description.

Click “Save” to store the new diagnosis.
